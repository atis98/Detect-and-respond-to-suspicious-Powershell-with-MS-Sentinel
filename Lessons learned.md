# This test attack and response provided the following key takeaway points:
- Devices that are used to interact with sensitive data and applications should always be onboarded to an endpoint detection and response system like Defender for Endpoint.
- data lookup queries written in Kusto Query Language are a powerful tool to retrieve and filter out specific data we need from huge databases.
- a SIEM-SOAR system like MS Sentinel, connected with an endpoint detection system and other security tools, is a very strong combination in order to receive alerts of potentially malicious events and to create an effective response mechanism to them
- MS Sentinel Playbooks triggered by Logic apps are useful for creating an automatic threat response mechanism, either by connecting pre-built logic apps or building our own one to our own needs.
